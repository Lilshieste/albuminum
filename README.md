# Albuminum

Photo album creation and management app built with Phoenix LiveView.

## Setup

```bash
mix setup          # Install dependencies and setup database
mix phx.server     # Start server at http://localhost:4000
```

## Development

```bash
mix test           # Run tests
mix test --watch   # Run tests in watch mode (requires mix_test_watch)
iex -S mix phx.server  # Start server with interactive shell
```
